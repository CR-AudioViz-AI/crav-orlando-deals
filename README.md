# Orlando Trip Deal

Free Orlando vacation planning tools powered by Javari AI.

- **Production:** https://orlandotripdeal.com
- **Preview:** https://crav-orlando-deals.vercel.app

## Features
- Hotel comparison tools
- Car rental helpers
- Theme park ticket guides
- Disney Annual Pass calculator
- Javari AI chat assistant

## Tech Stack
- Static HTML/CSS/JS
- Vercel Serverless Functions (Node.js)
- Supabase (database)
